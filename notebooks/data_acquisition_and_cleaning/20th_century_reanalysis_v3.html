NOAA/CIRES/DOE 20th Century Reanalysis version 3¶

Types de données disponibles¶

Les données de surface sont les suivantes :¶

  • apcp (précipitation totale)
  • cape (convective available potential energy)
  • tmin (température minimale)
  • tmax (température maximale)
  • eatm (Cloud water content)
  • crain (Categorical Rain)
  • pr_wtr (Precipitable Water for entire atmosphere)

Les données de pressions sont les suivantes :¶

  • hgt (Geopotential height)
  • air (Air temperature)
  • omega
  • rhum (Relative humidity (%)

Notez que omega et rhum ne sont pas disponibles de 0 à 100 mb.

base_url = 'https://psl.noaa.gov/thredds/dodsC/Datasets/20thC_ReanV3/'

urls = []
for year in range(1836, 2016):
    if year<1981:
        liste_vars = {'hgt' : 'prsSI/hgt.{}.nc'.format(year),
                      'air' : 'prsSI/air.{}.nc'.format(year),
                      'omega' : 'prsSI/omega.{}.nc'.format(year),
                      'rhum' : 'prsSI/rhum.{}.nc'.format(year)}
    else:
        liste_vars = {'hgt' : 'prsMO/hgt.{}.nc'.format(year),
                      'air' : 'prsMO/air.{}.nc'.format(year),
                      'omega' : 'prsMO/omega.{}.nc'.format(year),
                      'rhum' : 'prsMO/rhum.{}.nc'.format(year)}
        
    for key, value in liste_vars.items():
        
        nb_time=100

        if calendar.isleap(year):
            nb_time=2927
        else:
            nb_time=2919
            
        if key in ['hgt','air']:
            files = [base_url + 
                     '{}?level[{}],lat[130:1:153],lon[264:1:308],time[0:1:{}],'.format(value, level, nb_time) +
                     '{}[0:1:{}][{}][130:1:153][264:1:308]'.format(key, nb_time, level) 
                     for level in [0,4,6,8,10,12,14,16,18,20,21,22,23,24,25,26,27]]
        elif key in ['omega','rhum']:
            files = [base_url + 
                     '{}?level[{}],lat[130:1:153],lon[264:1:308],time[0:1:{}],'.format(value, level, nb_time) +
                     '{}[0:1:{}][{}][130:1:153][264:1:308]'.format(key, nb_time, level) 
                     for level in [0,4,6,8,10,12,14,16,18,20]]
            
            cr20.reduce_netcdf(files, output_path, key, year)
